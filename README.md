# Lampara Wifi
Control de luces Neopixel manejado desde un ESP que lanza una web desde la que seleccionar el color en el que colocar las luces.

## Hardware

La implementación más sencilla es utilizar un WifiPixels que incluye un ESP8266 con un anillo de 16  Led Addressable RGB: 

![WifiPixels](http://wiki.protoneer.co.nz/images/c/c6/WifiPixels_Top.JPG)

Pero si quieres también puedes montarlo tu mismo, necesaitaras los siguentes materiales:

* ESP2866 (En cualquiera de sus versiones)
* Leds Addresables RGB

Dependiendo del modulo que uses quizás necestes también necesitarás un convertidor FTDI - USB, a no ser que trabajes con un NodeMCU.

### Montaje

_SECCIÓN EN CONTRUCCIÓN_

## Software

El skecth que lleva cargada la lampara lanza una web al conectarse a la IP de la lamapara. Desde esta web puede seleccionarse el color que queramos y la intensidad desde cualquier dispositivo que sea capaz de lanzar un navegador.
